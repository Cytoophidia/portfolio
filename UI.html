Page NO.2

EySend

The interface you see here is that of a cutting-edge app that streamlines the money transfer process, making it easier and more efficient for users to send money to their intended recipients.

Inspiration from 'Venmo'

The interface allows users to easily view and select recent recipients, choose from multiple bank cards stored in their wallet, specify the transfer amount, and review all the relevant information. The app features a flat UI design, which incorporates interactive elements such as hover boxes and frame overflow sliding to enhance the user experience.
